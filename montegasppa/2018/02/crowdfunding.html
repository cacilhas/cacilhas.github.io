Reflexões… :: Vaquinha

24 de Fevereiro, 2018
Como um hipopótamo lida com um campo de flores.

Iniciei uma campanha de crowdfunding no Kickante, que vai até dia 22 de abril.



Preciso levantar fundos para adquirir alguns equipamentos que preciso para dar prosseguimento às gravações do projeto Prognose:

  • Um pedal em substituição a meu velho, de efeitos muito artificial.
  • Um violão de nylon com melhor captação que meu atual.
  • Um amphead para meu talkbox.
  • Um launchpad para controlar os soundscapes.
  • Uma mesa de som.

Apesar de eu precisar realmente de fundos para comprar os equipamentos, o objetivo principal desta campanha não é esse, mas testar o potencial de financialmente desse tipo de abordagem.

Se funcionar, pretendo fazer outra vaquinha para bancar os custos da gravação do projeto de fato, dessa vez com recompensas para os patrocinadores.

Caso alguém se sinta altruísta e benevolente (espero), pode colaborar com minha campanha neste link.
